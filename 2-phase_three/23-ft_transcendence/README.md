# A Pong Game

## Como rodar

> [!WARNING] Atenção
> - Verificar se as pastas de volumes do docker compose para o banco de dados estão criadas.
> - Verificar o arquivo .env
> - Verificar se o git submodules funcionou
> - Aplicar migrations

```sh
make
```

Reiniciar containers
```sh
make att
```
